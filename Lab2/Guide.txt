Для запуску програми необхідно встановити Docker. Після закинути csv з даними ЗНО 19-го та 21-го року у папку src. 
Далі з терміналу виконати команду docker-compose up --build (у фалі db.ini у полі host вказати 'db' для запуску з докеру, або 'localhost' для локального запуску).
Lab2.py містить основний код програми.
У папці flyway/sql знаходяться файли міграцій.
Результати виконання sql запиту записуються у файл zno_results.csv
