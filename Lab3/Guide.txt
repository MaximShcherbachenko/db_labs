Усі необхідні матеріали для розгортки сайту на heroku присутні у проекті.
Для власноручного розгортання потрбіно:
  1. Створити новий аплікейшн у совему профілі heroku;
  2. Під'єднатися до проекту (у вікні deploy аплікейшну можна знайти детальну інструкцію до цього);
  3. Запушити проект на main гілку heroku (деталі також у вікні deploy);
  4. З терміналу запустити heroku run python і виконати дві команди для створення бази даних:
    - from app import db
    - db.create_all()
Після виконання третього пункту heroku надає посилання на розгорнутий проект.
Наразі проект можна знайти за наступним посиланням:
https://flask-lab3.herokuapp.com/
